# 喵玉殿新版论坛

开发文档： <https://root.nsdn.club/nya-develop-document/>

## 技术栈

|技术栈|功能|
|-|-|
|vite|项目工具|
|vue3|JS 框架|
|TypeScript|编程语言|
|vue-router|路由|
|pinia|store|
|axios|HTML 请求|
|Naive UI|UI 框架|
|tiptap|富文本编缉器|
|marked|Markdown 编缉器|

## 启动项目

### 安装依赖

```shell
npm install
```

### 启动本地开发服务

```shell
npm run dev
```
